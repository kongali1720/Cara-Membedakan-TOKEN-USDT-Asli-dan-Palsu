# 🔐 Security Policy

## 📢 Reporting Vulnerabilities

Jika menemukan bug atau celah keamanan:

- Jangan publish ke publik
- Laporkan melalui Issues (private jika perlu)

## 🛡️ Scope
Project ini fokus pada:
- Edukasi crypto
- Verifikasi token
- Anti scam awareness

## ⚠️ Disclaimer
Project ini hanya untuk edukasi, bukan untuk aktivitas ilegal.
